# Install

1) Close Qt Creator.
2) Copy the contents of the XML files in ExternalTools/externaltools to: /home/user/.config/QtProject/qtcreator/externaltools
3) Locate file /home/user/.config/QtProject/QtCreator.ini, modify the [ExternalTools] argument with the contents of file ExternalTools/QtCreator.ini_externaltools, and save.
4) Open Qt Creator again and go to Edit > Preferences > Environment > External Tools and check the changes.

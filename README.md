2. **Cpp**
   - Códigos simples.

   - Arquivos de ajuda para o ambiente OpenBox e Debian.
4. **Sublime_Text**
   - **Sublime-Build**

5. **Python**
   - **BD_Treino:** Gerar SQL para treino de banco de dados.

   - **csv2vcf:** Faz a conversão de arquivo csv em vcf para exportação de contatos.
